# 🎮 AI像素冒险 - 口袋妖怪风格RPG游戏

> 一个用纯前端（HTML5 Canvas + JavaScript）打造的口袋妖怪风格像素RPG游戏
> **单HTML文件，双击即玩，零依赖**

[![GitHub Stars](https://img.shields.io/github/stars/jerry0509/pixel-rpg?style=social)](https://github.com/jerry0509/pixel-rpg)

---

## 🎯 游戏特色

- 🗺️ **8张地图** - 新手村、翡翠草原、幽暗森林、烈焰火山、魔王城堡、迷雾沼泽、远古遗迹、龙之巢穴
- ⚔️ **30+种怪物** - 史莱姆、狼、蝙蝠、骷髅、火蜥蜴、岩石怪...每种都有独特属性
- 🐉 **7个Boss** - 野猪王→狼王→火龙→魔王→沼泽之王→远古巨像→远古巨龙
- 🎨 **像素精灵系统** - 16x16/32x32像素画，36色调色板
- 💥 **属性相克** - 火→冰→雷→水→火，光暗互克
- 🌙 **昼夜循环** - 白天/黄昏/夜晚/黎明，影响遇敌率
- 🌧️ **天气系统** - 晴天/雨天/雪天/雾天
- 💾 **存档系统** - LocalStorage本地存档
- 🔊 **音效系统** - Web Audio API生成

## 🏗️ 技术架构

```
index.html（约2000行）
├── 配置常量（CFG）
├── 数据定义（技能、物品、怪物、NPC、地图）
├── 输入系统（Input）
├── 像素精灵系统（SpriteSheet）    ← 核心创新
├── 渲染引擎（Renderer）
├── 地图系统（MapMgr）
├── 玩家系统（Player）
├── 怪物类（Monster）
├── 战斗系统（Battle）
├── UI系统（UI）
├── 存档系统（Save）
├── 音频系统（Audio）
└── 游戏主循环（Game Loop）
```

### 像素精灵系统

每个精灵用十六进制字符串编码（1字符=1像素，36色调色板）：

```javascript
// 草地瓦片（16x16 = 256字符）
'7777777777777777' +
'7787777777877777' +
'7777797777777977' +
...

// 初始化时解码到offscreen canvas
// 游戏循环只做drawImage，零计算开销
```

### 战斗动画（口袋妖怪风格）

```
玩家选择攻击 → 角色冲向怪物 → 怪物闪红+屏幕震动 → 角色退回
```

## 🛠️ 开发相关

### 项目文件

```
pixel-rpg/
├── index.html          ← 游戏主文件（约2000行）
├── GAME_DESIGN.md      ← 完整游戏设计文档
├── BUGS_AND_ISSUES.md  ← 问题清单
├── PLAN.md             ← 开发计划
└── js/                 ← 旧版多文件代码（备用）
```

### 技术栈

- **前端**：HTML5 Canvas + JavaScript ES6+
- **渲染**：64x64像素瓦片，60fps
- **存储**：LocalStorage
- **音效**：Web Audio API
- **架构**：单HTML文件，零依赖
